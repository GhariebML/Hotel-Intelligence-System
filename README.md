# Hotel Booking Intelligence Dashboard

An interactive, premium Exploratory Data Analysis (EDA) dashboard developed for the **Data Mining & Visualization** course at **MTC**.

## 🚀 Overview
This project transforms a dataset of over **119,000 hotel bookings** into a professional, real-time analytics platform. It highlights key business insights, cancellation risk factors, and guest behavioral patterns through interactive visualizations.

### Key Features
- **Dynamic Data Engine**: Real-time CSV parsing (using PapaParse) of `hotel_booking.csv`.
- **High-Fidelity Visuals**: Advanced Chart.js implementations including Mixed Trend Line/Bar charts, Seasonal Heatmaps, and interactive Doughnuts.
- **Glassmorphism UI**: A modern, dark-themed responsive interface with animated background blobs.
- **Machine Learning**: Integrated insights from Random Forest classifiers and K-Means clustering.
- **Database Architecture**: Full Conceptual and Logical ER Diagrams.

## 📖 How to Run Locally
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the directory:
   ```bash
   cd DM_App
   ```
3. Start a local server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   ```
4. Open your browser to `http://localhost:8000`.
